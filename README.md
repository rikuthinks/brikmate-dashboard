# Brikmate Dev Application

Contains the ingestion API endpoint and upload functionality for the Brikmate dashboard.

Warning: Ingestion will not work once the OpenAI API usage has hit its limit.

## Development

```
npm install
```

Start dev

```
npm run dev
```
